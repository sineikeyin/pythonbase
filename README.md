# 🐍 Python 学习笔记

本项目是一个 **Python 零基础入门** 的学习笔记集合，使用 Jupyter Notebook 记录，涵盖从基础语法到面向对象编程的核心内容。

## 📂 文件结构

| 文件 | 说明 |
|:-----|:-----|
| [`python入门.ipynb`](python入门.ipynb) | Python 基础入门：print / input 函数、变量、格式化输出、关键字等 |
| [`六大数据类型.ipynb`](六大数据类型.ipynb) | Python 六大内置数据类型详解：数值、字符串、列表、字典、元组、集合 |
| [`python三大程序结构.ipynb`](python三大程序结构.ipynb) | 三大程序结构：顺序、分支（if/elif/else）、循环（while/for） |
| [`面向对象编程.ipynb`](面向对象编程.ipynb) | 面向对象编程：类与对象、封装、继承、多态 |

## 📖 内容概览

### 1. Python 入门基础
- `print()` 输出函数（分隔符、换行、文件输出）
- `input()` 输入函数
- 变量与对象（id、type、value）
- 格式化输出（`%`、`format()`、`f-string`）
- 变量的命名规则与关键字

### 2. 六大数据类型
- **数值类型**：整数、浮点数、复数
- **字符串**：索引、切片、常用方法
- **列表**：增删改查、列表推导式
- **字典**：键值对操作、常用方法
- **元组**：不可变序列、拆包
- **集合**：去重、集合运算

### 3. 三大程序结构
- **顺序结构**：程序按顺序逐行执行
- **分支结构**：`if` / `elif` / `else` 条件判断
- **循环结构**：`while` 循环、`for` 循环、`break` / `continue`

### 4. 面向对象编程（OOP）
- 类与对象的定义及使用
- 三大特性：封装、继承、多态

## 🚀 如何使用

### 前置条件
安装 [Python](https://www.python.org/)（建议 3.8+）和 [Jupyter Notebook](https://jupyter.org/)：

```bash
pip install jupyter
```

### 启动
在项目目录下运行：

```bash
jupyter notebook
```

然后在浏览器中打开对应的 `.ipynb` 文件即可查看和学习。
